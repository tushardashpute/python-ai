## 📚 Course Structure

This course is organized into **3 weeks**, with **2 classes per week**. Each class builds on the previous one, creating a progression from **understanding → controlling → augmenting AI**.

```
Week 1: UNDERSTANDING AI
├─ Day 1: How LLMs actually work
└─ Day 2: Prompt systems & architecture

Week 2: CONTROLLING AI BEHAVIOR
├─ Day 1: Structured output (JSON)
└─ Day 2: Advanced pipelines & refinement

Week 3: AUGMENTING AI WITH KNOWLEDGE
├─ Day 1: RAG foundations
└─ Day 2: Embeddings & semantic search

By Week 3 End: Production-ready AI systems!
```

## 💡 Core Concepts Across the Course

These ideas repeat and deepen:

1. **System Thinking:** Always think model + code + orchestration
2. **Validation:** Never trust AI output without verification
3. **Decomposition:** Complex tasks = Multiple simple steps
4. **Grounding:** Give AI context, prevent hallucination
5. **Iteration:** Refinement loops > Single shots
6. **Production Mindset:** Assume things will fail; plan accordingly
