## 🛠️ Adding Custom Errors

To add a new error pattern, edit `error_resolution_assistant.py`:

```python
knowledge_base["your_error_key"] = {
    "error_name": "Your Error Name",
    "description": "What this error means",
    "resolution": "How to fix it (concise)",
    "commands": [
        "kubectl command 1",
        "kubectl command 2",
        "kubectl command 3"
    ]
}
```

### Example: Adding Disk Pressure Error
```python
knowledge_base["disk_pressure"] = {
    "error_name": "Node Disk Pressure",
    "description": "Node running out of disk space",
    "resolution": "Clean up old logs/images or add storage",
    "commands": [
        "df -h",
        "kubectl describe node <node-name>",
        "sudo rm -rf /var/log/*"
    ]
}
```
